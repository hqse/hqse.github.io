WINCE版OziExplorer中一些英文菜单命令的含义

下面把WINCE版OziExplorer软件中常用的一些英文菜单命令整理出来供大家参考：

“File” --> “Waypoints” --> “Import Wp's from File” 导入航点文件
“File” --> “Waypoints” --> “Export Wp's to File” 导出航点文件
“File” --> “Waypoints” --> “Delete All Wp's” (从地图上)删除所有航点
“File” --> “Route” --> “Load Route File” 导入路径文件
“File” --> “Route” --> “Save Route File” 保存路径文件
“File” --> “Route” --> “Clear Route from Map” 从地图上清除路径
“File” --> “Track Log” --> “Log Track to File” 把轨迹日志保存到文件中
“File” --> “Track Log” --> “Copy Track Log File” 复制轨迹日志文件
“File” --> “Track Log” --> “Delete Track Log File” 删除轨迹日志文件
“File” --> “Track Log” --> “Import Track Tail from File” 导入轨迹文件
“File” --> “Track Log” --> “Export Track Tail to File” 导出轨迹文件
“File” --> “Track Log” --> “Clear Track Tail” (从地图上)清除轨迹
“File” --> “Lock Screen” 锁屏(锁屏后将滑块滑向右边可以解锁)

“View” --> “Waypoints” --> “Show” (勾选)显示航点(否则不显示航点)
“View” --> “Waypoints” --> “Highlight Waypoints” (勾选)高亮显示航点
“View” --> “Waypoints” --> “Show Names” (勾选)显示航点名称
“View” --> “Route” --> “Show” (勾选)显示路径(否则不显示路径)
“View” --> “Route” --> “Highlight Route” (勾选)高亮显示路径
“View” --> “Route” --> “Show WP Names” (勾选)显示航点名称
“View” --> “Route” --> “Show Route Name” (勾选)显示路径名称
“View” --> “Tracks” --> “Show” (勾选)显示轨迹(否则不显示轨迹)

“Map” --> “Connect to GPS” (勾选)连接到GPS
